HOW TO USE THE AI SoW GENERATOR (NO CODING NEEDED)

1. Replace your OpenAI key in 'Run_Me.bat' file:
   - Open 'Run_Me.bat' in Notepad
   - Replace 'sk-REPLACE_THIS_WITH_YOUR_KEY' with your real OpenAI key from https://platform.openai.com/account/api-keys
   - Save and close

2. Double-click 'Run_Me.bat'
   - A command window will open
   - Your browser will launch the app at http://localhost:8501

3. Use the app:
   - Upload PDF/DOCX contract
   - Enter description and pick examples
   - Click 'Generate'
   - Download the SoW as DOCX

Need help? Ask ChatGPT for step-by-step support.
